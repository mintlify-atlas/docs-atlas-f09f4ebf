## Core tools

The base image includes essential development utilities:

- `curl`, `wget`, `jq`, `unzip`
- `git`, `lsof`, `htop`
- `sudo`, `bash`, `bash-completion`
- `ca-certificates`, `gnupg`, `openssh-client`
- `build-essential`, `gawk`, `procps`

## Git credential helper

Includes `git-credential-libsecret` built from Git source v2.52.0 for secure credential storage.

## SSH configuration

OpenSSH server is included with ET-based resilient SSH transport support. When enabled, services run on loopback:

- `etserver` on `127.0.0.1:2022`
- `sshd` on `127.0.0.1:2244`

## User setup

Default user is `coder` (UID 1000) with:
- Passwordless sudo access
- Docker group membership
- Bash shell with completion
