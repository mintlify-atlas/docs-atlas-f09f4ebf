## VS Code extensions

The following extensions are pre-configured:

- `JakeBecker.elixir-ls` - Elixir language server
- `phoenixframework.phoenix` - Phoenix framework support
- `bradlc.vscode-tailwindcss` - Tailwind CSS IntelliSense
- `dbaeumer.vscode-eslint` - ESLint integration
- `esbenp.prettier-vscode` - Prettier code formatter

## VS Code settings

Pre-configured settings for Elixir and Phoenix development:

- **Elixir files**: Auto-format on save, 2-space indentation
- **Phoenix HEEX templates**: Auto-format on save with Elixir LS
- **Emmet**: Enabled for HEEX and Elixir files with HTML completion

## Base tooling

Inherits all tools from the [base image](/reference/images/base), including:

- Docker CLI, Compose, and Buildx
- Coder agent and code-server
- Chrome for Testing and ChromeDriver
- Mise with common CLI tools (ripgrep, fd, fzf, etc.)
