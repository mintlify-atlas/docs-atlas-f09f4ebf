</ParamField>

<ParamField path="Profile" type="string">
  minimal
</ParamField>

The minimal profile includes `rustc`, `cargo`, and `rustfmt`, but excludes documentation and other optional components to reduce image size. You can add additional components using `rustup component add` as needed.

## Editor tooling

<ParamField path="LazyVim" type="string">

Pre-configured Neovim distribution with IDE-like features.

## VS Code extensions

The following extensions are pre-configured:

- `rust-lang.rust-analyzer` - Rust language server
- `tamasfe.even-better-toml` - TOML language support
- `vadimcn.vscode-lldb` - Native debugger for Rust

## Base tooling

Inherits all tools from the [base image](/reference/images/base), including:

- Docker CLI, Compose, and Buildx
- Coder agent and code-server
- Chrome for Testing and ChromeDriver
- Mise with common CLI tools (ripgrep, fd, fzf, etc.)
